# mojie

## 概要

これは写真をアスキーアートに変えるためのプログラムです

## インストール

1. [pip install -r requirements.txt]


## 使い方

1. [起動したら出てくる画面で"画像をアスキーにする"を選択]
2. ['画像ファイルを開く'と書いてあるボタンを押して任意の画像を選択]
3. ['アスキ-に変換'を押して変換]
4. ['コピー'を押してクリップボードにコピー]
