# mojie

## 概要

これは写真をアスキーアートに変えるためのプログラムです

## インストール

1. [pip install -r requirements.txt]


## 使い方

1. [起動したら出てくる画面で画像をアスキーに変換を選択]
2. [画像を選択と書いてあるボタンを押して任意の画像を選択]
3. [アスキーに変換を押して変換]
